## Description

A multi-project application

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start iam|aod|todo

# watch mode
$ npm run start:dev iam|aod|todo

# production mode
$ npm run start:prod iam|aod|todo
```

## Test

```bash
# unit tests
$ npm run test iam|aod|todo

# e2e tests
$ npm run test:e2e iam|aod|todo

# test coverage
$ npm run test:cov iam|aod|todo
```
